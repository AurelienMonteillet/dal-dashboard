Tezos DAL-o-meter Data API

This page serves data for the Tezos DAL-o-meter Dashboard. The data is available in JSON format at the following URLs:

Current DAL Statistics:
dal_stats.json

Historical DAL Statistics:
dal_stats_history.json

These endpoints are updated periodically to reflect the latest data from the Tezos blockchain.

Visit the GitHub repository for more information about the DAL-o-meter project.
